# WorkplaceFabricsFreespace

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 6.1.3.


## Steps to run the application
* Clone the repo
* _run_ ` npm install`
* _run_ `ng serve --open`. It should auto navigate to `http://localhost:4200/` in your default browser. If that doesn't happens automatically, you can manually navigate.
* To specify port and host while running the application pass `--port <port>` and `--host <host>` into ng serve command 
  Example - `ng serve --open --port 4200 --host 192.167.1.1`

## Steps to use the application
* Select any block from the floor plan to view its details
* To update any desk click edit button and update its editable values
* Click Save button to update the floor plan
* You can refresh appplication using F5 key and the updated floor plan remains consistent

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.


### Assumptions

SVG image consists the floor plan 
and each rectangle represents a desk.
